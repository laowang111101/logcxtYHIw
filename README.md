# 图书管理系统 python1058

## 项目概述

本项目是一个基于Python语言和SQLite数据库的图书管理系统。系统主要分为用户和管理员两个角色，致力于实现书籍信息的有效管理。

## 技术栈

- 后端：Python
- 数据库：SQLite

## 功能模块

### 用户模块

- 查看书籍信息

### 管理员模块

- 书籍信息的增加
- 书籍信息的删除
- 书籍信息的修改
- 书籍信息的查询

## 系统角色

- 用户：可以查看书籍信息。
- 管理员：可以对书籍信息进行增删改查操作。

## 运行环境

- 操作系统：兼容Windows、Linux和MacOS
- Python版本：3.6及以上
- SQLite版本：3.28及以上

## 部署步骤

1. 安装Python环境。
2. 安装SQLite数据库。
3. 克隆项目代码到本地。
4. 运行数据库脚本以建立数据库结构。
5. 启动项目。

## 目录结构

```
book_management_system/
├── app/
│   ├── admin/                # 管理员模块代码
│   └── user/                 # 用户模块代码
├── database/                 # 数据库文件
│   └── book.db               # SQLite数据库
├── scripts/                  # 数据库脚本
└── main.py                   # 程序主入口
```

## 核心亮点

- 使用Python简洁的语法，易于维护和扩展。
- 基于SQLite，无需复杂的数据库配置，方便部署和使用。
- 清晰的角色划分，确保系统操作的权限控制。
